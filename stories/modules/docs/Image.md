
# Image Module

## Example
```jsx
{
  name: 'image',
  type: 'image',
  src: "https://portal.gems.org/images/complete-tasks.png",
}
```

## Image properties

| Property | type    | Required | Default | description |
| ---------| ------- | -------- | ------- | ----------- |
| *type*   | 'video' | true     | -       | module type |
| *name*   | string  | true     | -       | module name |
| *src*    | string  | true     | -       | video url   |
