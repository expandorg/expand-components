# Iframe Module

## Iframe properties

| Property | type    | Required | Default | description |
| ---------| ------- | -------- | ------- | ----------- |
| *type*   | `iframe` | true     | -       | module type |
| *name*   | string  | true     | -       | module name |
| *src*    | string  | true     | -       | iframe url   |
| *title*  | string  | false    | -       | iframe title   |

## Example
```jsx
{
  name: 'iframe',
  type: 'iframe',
  src: "https://expand.org/images/image-annotation.png",
}
```
