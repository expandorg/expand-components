# Verify

## Select properties

| Property       | type           | Required | Default | description  |
| -------------- | -------------- | -------- | --------| ------------ |
| *type*         | `verify`       | true     | -       | module type  |
| *name*         | string         | true     | -       | module name  |
| *yesCaption*   | string         | true     | -       | yes caption  |
| *noCaption*    | string         | true     | -       | no caption   |

## Example
```jsx
{
  name: 'verify',
  type: 'verify',
  yesCaption: 'Yes',
  noCaption: 'No',
}
```
