# Wizard Module

## Wizard properties

| Property     | type           | Required | Default | description |
| ------------ | -------------- | -------- | ------- | ----------- |
| *type*       | `wizard`       | true     | -       | module type |
| *name*       | string         | true     | -       | module name |


## Example
```jsx
{
  name: "wizard",
  type: "wizard",
  modules: [{
    name: "richText",
    type: "richText",
    content: "<h3>Step1</h3>"
  }, {
    name: "richText2",
    type: "richText",
    content: "<h3>Step2</h3>"
  }]
}

```
