# Progress Module


## Progress properties

| Property     | type           | Required | Default | description |
| ------------ | -------------- | -------- | ------- | ----------- |
| *type*       | `progress`     | true     | -       | module type |
| *name*       | string         | true     | -       | module name |
| *number*     | number         | true     | -       | current step |
| *total*      | number         | true     | -       | total number of steps |

## Example
```jsx
{
  name: "progress",
  type: "progress",
  number: 3,
  total: 10,
}
```
