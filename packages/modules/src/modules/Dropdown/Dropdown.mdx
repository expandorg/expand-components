# Dropdown Module

## Dropdown properties

| Property       | type           | Required | Default | description  |
| -------------- | -------------- | -------- | --------| ------------ |
| *type*         | `dropdown`       | true     | -       | module type  |
| *name*         | string         | true     | -       | module name  |
| *label*        | string         | false    | -       | label        |
| *options*      | `[ string, string... ]`   | []      | -     | array of options   |

## Example
```jsx
{
  name: 'dropdown',
  type: 'dropdown',
  label: 'test',
  options: ['Option 1', 'Option 2', 'Option 2', 'Option 4'],
}
```
