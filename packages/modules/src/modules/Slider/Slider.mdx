# Slider Module

## Slider properties

| Property       | type           | Required | Default | description |
| -------------- | -------------- | -------- | ------- | ----------- |
| *type*         | `slider`       | true     | -       | module type  |
| *name*         | string         | true     | -       | module name  |
| *min*          | number         | -        | -       | Min value  |
| *max*          | number         | -        | -       | Max  value |
| *step*         | number         | -        | -       | Step size |


## Example
```jsx
{
  name: 'slider',
  type: 'slider',
  min: 0,
  max: 100,
  step: 1,
}
```
