# Submit Module

## Submit properties

| Property  | type     | Required | Default | description |
| --------- | -------- | -------- | ------- | ----------- |
| *type*    | `submit` | true     | -       | module type |
| *name*    | string   | true     | -       | module name |
| *caption* | string   | true     | -       | button text |

## Example
```jsx
{
  name: 'submit',
  type: 'submit',
  caption: 'Next'
}
```
