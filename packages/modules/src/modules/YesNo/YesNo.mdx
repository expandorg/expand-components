# Yes/No

## Select properties

| Property       | type           | Required | Default | description  |
| -------------- | -------------- | -------- | --------| ------------ |
| *type*         | `select`       | true     | -       | module type  |
| *name*         | string         | true     | -       | module name  |
| *idType*       | 'none , `small`, `capital`, `roman`, `numerals` | false    | numerals       | enumerator type  |
| *yesCaption*   | string         | true     | -       | yes caption  |
| *noCaption*    | string         | true     | -       | no caption   |

## Example
```jsx
{
  name: 'yesno',
  type: 'yesno',
  idType: 'small',
  yesCaption: 'Yes',
  noCaption: 'No',
}
```
