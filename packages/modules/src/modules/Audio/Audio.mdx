# Audio

## Audio properties

| Property    | type    | Required | Default | description    |
| ----------- | ------- | -------- | ------- | -------------- |
| *type*      | `audio` | true     | -       | module type    |
| *name*      | string  | true     | -       | module name    |
| *src*       | string  | true     | -       | audio url      |
| *loop*      | bool    | -        | -       | loop audio     |


## Example
```jsx
{
  name: 'audio',
  type: 'audio',
  src: "https://soundcloud.com/dalyaka/ubud",
}
```
