# Image Module

## Image properties

| Property | type    | Required | Default | description |
| ---------| ------- | -------- | ------- | ----------- |
| *type*   | `image` | true     | -       | module type |
| *name*   | string  | true     | -       | module name |
| *src*    | string  | true     | -       | image url   |

## Example
```jsx
{
  name: 'image',
  type: 'image',
  src: "https://expand.org/images/image-annotation.png",
}
```
