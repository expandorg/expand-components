# RichText Module

## RichText properties

| Property     | type           | Required | Default | description |
| ------------ | -------------- | -------- | ------- | ----------- |
| *type*       | `richText`    | true     | -       | module type |
| *name*       | string         | true     | -       | module name |
| *content*    | string         | -        | -       | html markup        |


## Example
```jsx
{
  name: "richText",
  type: "richText",
  content: "<h3>Lorem ipsum dolor sit amet</h3>"
}
```
