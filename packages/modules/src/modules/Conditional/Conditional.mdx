## Show/Hide modules conditionally

## Example
```jsx
{
  name: "checkbox",
  type: "checkbox",
  label: "select me",
},
{
  name: "input",
  type: "input",
  placeholder: 'type something',
  inputType: 'text',
  logic: {
    show: ['$(checkbox)', '==', 'true']
  }
}
```
