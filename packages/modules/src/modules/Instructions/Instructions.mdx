# Instructions Module


## Instructions properties
 properties

| Property     | type           | Required | Default | description |
| ------------ | -------------- | -------- | ------- | ----------- |
| *type*       | `instructionsItem` | true | -       | module type |
| *name*       | string         | true     | -       | module name |
| *button*     | string         | false    | -       | trigger button |
| *modules*   | `module`        | true     | -       | dialog content |


## Example
```jsx
{
  type: 'instructions',
  name: 'instructions',
  button: 'Instructions',
  modules: [
    {
      name: 'header',
      type: 'text',
      style: 'body',
```
