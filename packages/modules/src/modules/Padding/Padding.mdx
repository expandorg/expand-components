# Padding Module

## Padding properties

| Property     | type           | Required | Default | description |
| ------------ | -------------- | -------- | ------- | ----------- |
| *type*       | `padding`      | true     | -       | module type |
| *name*       | string         | true     | -       | module name |

## Example
```jsx
{
  name: "padding",
  type: "padding",
}
```
