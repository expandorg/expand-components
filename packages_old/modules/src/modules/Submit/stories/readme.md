
# Submit Module

## Example
```jsx
{
  name: 'submit',
  type: 'submit',
  caption: 'Next',
  justify: 'left',
}
```

## Submit properties

| Property  | type     | Required | Default | description |
| --------- | -------- | -------- | ------- | ----------- |
| *type*    | `submit` | true     | -       | module type |
| *name*    | string   | true     | -       | module name |
| *caption* | string   | true     | -       | button text |
| *justify* | `left`, `right`, `center`, `between`  | -        | `center`       | horizontal alignment |
